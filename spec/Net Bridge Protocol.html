Net Bridge Protocol
John Haugeland, Jeff Katz$Revision$ The Net Bridge Protocol is an extremely simple reference layer for other protocols to be built on top of; its purpose is to bury the annoying details of network behavior in most languages. By design it is simple, message oriented, language portable and robust. The Net Bridge Protocol makes no attempt to handle complex issues like quality of service; it is merely a lightweight wrapper to allow a client and/or server implementation to be cranked out simply and re-usably in any given language.
