Description

Extended quickcheck generators. Kept separate because I don't want to require eQC to compile scutil, and because eQC conflicts eval/1,2 and shuffle/1. The list of QuickCheck generators could use substantial extension, and, well, here it is. These are essentially entirely used by sc_tests.erl.

non_neg_integer/0

non_neg_integer() -> any()

Generator to produce non-negative integers, with otherwise identical behavior to eqc_gen:int(). Inherits range scaling behavior from use of int() macro. May produce zeros - if you don't want zeros, use pos_integer() instead.

rand_elements/1

rand_elements(ElementList) -> any()
