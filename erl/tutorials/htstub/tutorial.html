How-to: HtStub Using an Erlang Application-Specific Webserver Library

HtStub is a webserver library. To be long-winded about it, "HtStub is an embeddable application-specific webserver library for Erlang." This document serves as a bootstrap and tutorial; there are formal docs elsewhere. This text is just to get people started, to help them understand what HtStub is about, how to use it and what makes it different.

HtStub is about one thing: taking the boring part out of Erlang webservers, by reducing the programmer workload to as close to zero as possible.
